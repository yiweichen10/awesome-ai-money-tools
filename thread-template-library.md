# Twitter Thread 模板库与首批选题

适用账号：@CHiang_AILab
定位：AI工具 + 在线赚钱方法 + 从0开始

## A. 10个 Thread 模板

### 1. 从0到1启动型
**标题模板**：If I had to make my first $100 with AI in the next 7 days, I'd do this
- Hook：限定时间 + 限定金额
- 中段：步骤1-5
- 收尾：强调只做一个方向

### 2. 反常识打脸型
**标题模板**：Most AI side hustles are a waste of time
- Hook：强判断
- 中段：为什么错 + 真正值得做的1-2个方向
- 收尾：The boring stuff makes money

### 3. 工具替代岗位型
**标题模板**：3 AI tools that can replace a $1,000/month freelancer
- Hook：替代岗位/成本
- 中段：每个工具负责什么
- 收尾：Would you build this solo?

### 4. 一人系统型
**标题模板**：A one-person AI income system in 5 steps
- Hook：系统感
- 中段：从获客到交付的完整流程
- 收尾：Simple > complex

### 5. 错误避坑型
**标题模板**：Why beginners fail with AI
- Hook：点出失败原因
- 中段：列出3-5个错误
- 收尾：Pick one offer, not five tools

### 6. 平台玩法型
**标题模板**：How I'd use Twitter + GitHub + AI to build income from $0
- Hook：平台组合
- 中段：内容、引流、转化闭环
- 收尾：This is slower than hype, but it compounds

### 7. 工具对比型
**标题模板**：ChatGPT vs Claude vs Jasper for making money
- Hook：明确对比对象
- 中段：按场景拆分，不做百科式介绍
- 收尾：Most people only need one of them

### 8. 客户交付型
**标题模板**：How to sell AI services without being an expert
- Hook：降低门槛
- 中段：如何选服务、如何交付、如何定价
- 收尾：You don't need to be elite. You need to be useful

### 9. 热点拆解型
**标题模板**：Why everyone is suddenly talking about [tool/trend]
- Hook：借热点
- 中段：解释热度原因 + 赚钱方式
- 收尾：Trend or real business? Here's my take

### 10. 清单转执行型
**标题模板**：Don't start 10 AI hustles. Start this one
- Hook：反对贪多
- 中段：把一个方向拆成今天就能做的动作
- 收尾：Execution beats research

## D. 模板使用规则

- 每个 thread 只选一个模板，不混搭太多结构
- 每个 thread 必须有一个具体场景
- 每个 thread 至少出现 3 个具体工具/动作/金额/时间细节
- 优先围绕当前账号已有资产：Twitter、GitHub、affiliate、电子书
