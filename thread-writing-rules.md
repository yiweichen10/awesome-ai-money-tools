# Twitter Thread 去AI味写作规则

适用账号：@CHiang_AILab  
定位：AI工具 + 在线赚钱方法 + 从0开始

## 1. 核心目标

Thread 不是“知识堆砌”，而是：
- 让读者感觉这是一位真的在做事的人写的
- 提供明确判断，而不是中性总结
- 提供可执行步骤，而不是空泛鼓励
- 引导收藏、转发、关注，但不过度营销

## 2. 去AI味原则

### 必须做到
1. **每个 thread 只有一个核心结论**
2. **开头必须有立场/反差/冲突**，不能只是“Here are 7 ways...”
3. **至少包含 3 个具体细节**：工具名、金额、时间、平台、步骤、限制条件
4. **句长不平均**：短句 + 中句 + 停顿句混合
5. **必须像人说话**：允许有“Honestly / Boring? Yes. / That's the play.”这类自然口语
6. **结尾优先用提问或判断收束**，不要机械地每次都“Follow for more”
7. **优先使用真实来源**：Reddit、TikTok、GitHub、Product Hunt、真实工具页、已发布推文数据

### 明确禁止
- 套话开头：
  - "In today's fast-paced digital world..."
  - "AI is revolutionizing..."
  - "Here are 10 ways to..."
  - "Let's dive in"
- 空洞判断：
  - "Consistency is key"
  - "AI is the future"
  - "The opportunities are endless"
- 过于均匀的结构：每条都同长度、同语法、同结尾
- 没有任何具体工具/金额/动作的“鸡汤型” thread
- 每条都带相同 CTA（Follow/Bookmark/RT）

## 3. 推荐结构

### 结构 A：反常识型
1. Hook：一个反直觉判断
2. 解释为什么大多数人错了
3. 给出 3-5 个具体原因/步骤
4. 总结一个更务实的做法
5. 结尾提问或一句强判断

### 结构 B：从0到1型
1. Hook：如果从 $0 开始，我会怎么做
2. 限定条件（时间/预算/技能）
3. 按顺序拆解步骤 1-5
4. 给出现实收入预期
5. 结尾强调“不要同时做5件事”之类的判断

### 结构 C：工具变现型
1. Hook：这个工具如何替代一个岗位/流程
2. 工具名 + 解决的问题
3. 使用流程
4. 怎么卖给客户 / 怎么自己变现
5. 金额与风险提醒
6. 结尾提问

### 结构 D：流程拆解型
1. Hook：一个完整系统/工作流
2. Step 1-5
3. 每一步说明使用什么工具
4. 适合什么人
5. 最后一句强结论

## 4. 语言风格要求

- 多用短句和断句
- 多用动作动词：pick, sell, ship, automate, test, repeat
- 少用抽象词：leverage, unlock, maximize, revolutionize, optimize（除非必要）
- 金额、时间、平台要尽量明确
- 允许不完全对称，让内容更像真人写的

## 5. CTA 规则

可用 CTA：
- "Which one would you try first?"
- "Most people overcomplicate this."
- "Want me to turn this into a checklist?"
- "The boring methods usually make the most money."
- "Would you rather build this or buy it?"

避免：
- 每条都用 "Follow for more"
- 每条都用 "Bookmark this"
- 每条都用 "RT if you agree"

## 6. 数据来源优先级

1. 当前账号已发布的高互动推文
2. Reddit 真实讨论
3. TikTok 热门趋势
4. 新AI工具/官网/Product Hunt
5. GitHub 仓库里的工具与方法

## 7. 自动生成时的质量检查

每个 thread 发布前必须检查：
- 是否有一个明确核心结论
- 是否至少有 3 个具体细节
- 是否避开套话开头
- 是否存在明显 AI 套词
- 每条是否低于 280 字符
- 结尾是否自然，不机械

## 8. 适合本账号的 Thread 类型

1. 从0到1：How I'd make my first $100 with AI
2. 反常识：Most AI side hustles are a waste of time
3. 工具变现：3 tools I'd use to replace a $1,000/month freelancer
4. 流程拆解：A one-person AI income system
5. 错误避坑：Why beginners fail with AI

## 9. 发布频率建议

- 每周 2 次高质量 thread
- 与短推文错开发布
- Thread 不追求量，追求收藏、转发、关注率

## 10. 最终标准

一条合格的 thread 应该让人感觉：
> “这不是AI在总结信息，而是一个真的在做这件事的人，在把自己筛选过的方法讲明白。”
